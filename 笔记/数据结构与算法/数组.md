## 1. 求斐波那契数列的前20项

斐波那契数列的每一项都是前面两项的和，其中第一项为1，第二项为2.

```js
var fibonacci = [];
[fibonacci[1], fibonacci[2]] = [1, 2];
for (let i = 3; i < 21; i++) {
    fibonacci[i] = fibonacci[i-1] + fibonacci[i-2];
}
for (let j = 1; j < 21; j++) {
    console.log(fibonacci[j]);
}
```
